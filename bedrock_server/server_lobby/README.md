# Lobby Server

Folder ini adalah server lobby utama untuk Minecraft Bedrock.

Fokus saat ini:
- lobby world untuk pemain
- titik awal sebelum masuk ke server lain
- titik spawn role di Maharlika City:
  - member: `-2068 33 -2043`
  - admin: `1600 31 -241`
- persiapan portal server di koordinat yang nanti kamu tentukan
- persiapan akses publik untuk pemain dari luar jaringan lokal
- batas kompatibilitas versi Bedrock yang realistis
