# Server_MC Gameplay Suite

Pack ini disiapkan sebagai kerangka fitur server:

1. Welcome fireworks dan chat saat player spawn pertama.
2. Barrier lobby yang nanti bisa diaktifkan dari `scripts/config.js`.
3. Guide book server.
4. Ekonomi koin dan penukaran item farming.
5. UI store dan placeholder NPC trigger.
6. Item phone untuk transfer item / koin antar player.

## Yang perlu kamu isi nanti

- Koordinat barrier lobby.
- Koordinat NPC store atau trade.
- Harga item store.
- Daftar item farming yang mau ditukar.

## Catatan

- Item "phone" dan "book" masih memakai item vanilla agar langsung aman dipakai.
- Untuk texture dan icon custom, kita bisa lanjutkan ke resource pack setelah alur gameplay final beres.
