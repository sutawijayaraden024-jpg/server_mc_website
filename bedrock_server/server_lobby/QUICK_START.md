# Quick Start — Lobby Server

1. Letakkan `bedrock_server.exe` di folder `server_lobby`.
2. Pastikan world lobby ada di `server_lobby\worlds\Maharlika_City`.
3. Jalankan `start_server.bat` untuk memulai server.
4. Buka port UDP `19132` pada router/VM jika ingin server dapat diakses publik.
5. Nanti, saat kamu memberi koordinat portal server, saya bisa bantu siapkan penanda/struktur untuk portal itu di world lobby.
6. Kalau targetnya akses publik, ikuti juga [Public Access Guide](PUBLIC_ACCESS.md).

Catatan:
- `bedrock_server.exe` tidak termasuk repository ini karena file biner.
- Pengaturan `server.properties` sekarang disetel untuk lobby yang aman dan siap dipakai.
- Kompatibilitas versi Bedrock itu sangat tergantung versi server yang dipakai. Secara praktik, server Bedrock biasanya paling aman dipakai pada versi client yang sama atau sangat dekat versinya.
