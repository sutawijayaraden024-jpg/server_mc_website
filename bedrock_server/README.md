# Bedrock Server Setup

Bedrock server untuk Minecraft Bedrock Edition.

## Struktur

```
bedrock_server/
├── lobby/                          # Main server directory
│   ├── start_server.bat           # Launch server script
│   ├── bedrock_server.exe         # Server executable
│   ├── server.properties          # Server configuration
│   ├── behavior_packs/            # Custom behavior packs
│   ├── resource_packs/            # Custom resource packs
│   ├── worlds/                    # World data
│   │   └── Maharlika_City/        # Main world
│   └── logs/                      # Server logs
├── plugins/                        # BDSX plugins
└── MaharlikaCityV.8.0.mcworld     # World backup

```
