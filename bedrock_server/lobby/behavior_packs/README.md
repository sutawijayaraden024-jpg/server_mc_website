# Behavior Packs

This directory contains installed behavior packs that modify game mechanics.

## About Behavior Packs

Behavior packs change how the game functions:
- Custom items and blocks
- Modified mobs and entities
- Game mechanics and rules
- Commands and functions

## How to Add

1. Copy behavior pack folder into this directory
2. Restart server
3. Add to world_behavior_packs.json

## Folder Structure

Each behavior pack should contain:
```
pack_name/
├── manifest.json
├── pack_icon.png
└── functions/
    └── (custom functions)
```
