# Resource Packs

This directory contains installed resource packs that modify textures and visuals.

## About Resource Packs

Resource packs change how the game looks:
- Custom textures
- Modified models
- New particle effects
- Custom fonts and UI

## How to Add

1. Copy resource pack folder into this directory
2. Restart server
3. Add to world_resource_packs.json

## Folder Structure

Each resource pack should contain:
```
pack_name/
├── manifest.json
├── pack_icon.png
└── textures/
    └── (texture files)
```
