Beranda Server Update Event Komunitas

Lobby & Respawn

Zona aman untuk respawn, belajar, dan memilih tujuan berikutnya dalam jaringan server kami.

Zona Aman

Area respawn bebas serangan, cocok untuk pemain baru yang ingin memahami sistem server.

Portal Akses

Portal di lobby memudahkan perpindahan ke server minigames, Immersial Worlds, dan server lainnya.

Version Support

Lobby mendukung Minecraft Bedrock 1.20, 1.21, dan versi 26.0 ke atas untuk koneksi stabil.

Fitur Lobby

  • Respawn point utama untuk semua pemain.
  • Informasi singkat tentang server dan event.
  • Pintu masuk ke area server lain dalam jaringan Server_MC.

Cara Akses

Masuk ke server utama lalu pilih jalur Login Lobby dari Hidden Hole. Lobby juga dapat diakses kembali saat respawn.
