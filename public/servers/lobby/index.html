Beranda Server Update Event Komunitas

Lobby & Respawn

Area aman di dalam Maharlika City untuk respawn, login akun server, membaca informasi, dan memulai perjalanan pemain.

Zona Aman

Area respawn bebas serangan, cocok untuk pemain baru yang ingin memahami sistem login dan arah bermain di Maharlika City.

Pusat Informasi

Lobby digunakan sebagai tempat menampilkan informasi server, akun, event, dan arahan sebelum pemain menjelajah world utama.

Version Support

Area ini mengikuti versi server utama Maharlika City dan disiapkan agar mudah dikembangkan bersama fitur server berikutnya.

Fitur Lobby

  • Respawn point utama untuk semua pemain.
  • Informasi singkat tentang server, akun, dan event.
  • Area awal sebelum pemain menjelajah Maharlika City.
  • Ruang transisi untuk member yang sudah login dan admin yang bertugas.

Cara Akses

Masuk ke server utama Maharlika City, login dengan akun website, lalu gunakan area lobby sebagai titik awal bermain.

Jika pemain belum login, sistem akun website akan membantu membedakan akses member dan admin sebelum fitur backend penuh diaktifkan.

Login Website

Gunakan akun yang sudah terdaftar untuk membuka akses member di server utama.

Buka Login

Register Akun

Kalau belum punya akun server, daftar dulu di website sebelum bermain sebagai member.

Buka Register

Status Singkat

Backend publik sudah disiapkan untuk sinkronisasi login. Saat website online, status akun akan tersambung otomatis ke API server.

Member

Member adalah pemain yang sudah register dan login di website. Setelah itu mereka bisa memakai fasilitas server sesuai role.

Admin

Admin adalah pengawas server yang menerima hasil farming/jual barang dan memantau pemain aktif melalui website.

Status Online

Ketika pemain login, statusnya dicatat sebagai online. Saat keluar, status tersebut otomatis dihapus dari daftar aktif.
