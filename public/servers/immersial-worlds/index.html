Beranda Server Update Event Komunitas

Immersial Worlds

Server eksplorasi utama dengan dukungan Minecraft Bedrock 1.20, 1.21, dan 26.0+.

Ukuran Dunia

World packed sekitar 335.32MB, dibuat untuk pengalaman petualangan besar dengan pemandangan luas dan teka-teki misteri.

Hidden Hole

Terletak jauh di dalam dunia, Hidden Hole menghadirkan 6 jalur: minigames, lobby respawn, server lain, dan 3 jalur buntu.

Gameplay

Dirancang untuk eksplorasi, misteri, dan discovery. Pemain harus menemukan portal tersembunyi untuk membuka server lain.

Fitur Utama

  • Petualangan dunia terbuka dengan nuansa teka-teki dan misteri.
  • Lingkungan yang dipengaruhi inspirasi NTE, GTA, dan anime bertema misteri.
  • Hidden Hole sebagai mekanik utama untuk akses server lain.
  • Kompatibel dengan Bedrock 1.20, 1.21, dan 26.0+.

Cara Bergabung

Tambahkan server utama kami di Minecraft Bedrock dengan alamat:

play.servermc.net

Setelah terhubung, mulailah menjelajah untuk menemukan Hidden Hole dan akses ke minigames, lobby, serta server lain.
