    Update 0.01 Beta

    Status: Sistem akun lokal sudah tersedia untuk tahap pengujian awal.

    Website sekarang mendukung register, login, role member dan admin, serta pencatatan pemain aktif lewat backend publik ketika situs dibuka online.

    Data akun dan sesi ditangani lewat API backend agar website bisa tetap public dan 24/7 tanpa bergantung pada browser lokal.
